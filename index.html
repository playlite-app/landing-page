Game Details

Everything about
a game, in one place.

The game details window now opens into four tabs: media, similar games, technical specs, and your personal notes. Data pulled live from PCGamingWiki — so you know exactly what you're installing before you hit download.

  • Screenshots and trailers in the Media tab
  • System requirements and language support from PCGamingWiki
  • Controller compatibility at a glance
  • Similar games recommended from your own library context

Everything you need

Built for people who take
their backlog seriously.

Every feature exists because managing a large game library actually needs it.

📦
Multi-store library

Import from Steam automatically. Add Epic, GOG, Xbox, or emulator games manually. One unified view.

📊
Status tracking

Mark games as Playing, Completed, Dropped, or Backlog. Rate them, favorite them, track playtime.

🌍
PT-BR & English

Full i18n support. The app detects your system language and switches automatically between Portuguese and English.

💾
Backup & restore

Export your entire library to JSON at any time. Restore it on a new machine in seconds.

🔄
Auto-updater

New releases delivered silently in the background. Cryptographically signed packages for integrity.

🔒
Private by design

No telemetry. No accounts. API keys stored locally in an encrypted database. Your data stays yours.
